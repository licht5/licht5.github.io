搭建hexo+github pages博客

简单说一下自己的搭建过程，系统为mac 10.10.

安装git

git链接

安装Node.js

node.js链接

安装hexo

利用npm命令，打开git bash
npm install -g hexo

创建hexo文件夹

安装完成后，在喜欢的文件夹如E:\blog下打开git bash，执行以下命令，Hexo即可生成建立网站所需要的所有文件
hexo init

安装依赖包

npm install

本地查看

hexo generate //保存修改
hexo server //开启本地服务器`

至此，本地博客搭建完毕，只能自己看到，下面部署到Github

注册Github

github链接

创建repository

用的是github pages ，可在此【github pages】 （https://pages.github.com/）
repository名字应该是licht5.github.io

部署

编辑_config.yml(在..\blog下)
deploy:
type: git
repository: https://github.com/txpfe/txpfe.github.io.git
branch: master
执行以下指令完成部署
hexo generate
hexo deploy //部署到github
当时出现了bug，so设置了ssh，请看这里
每次使用命令，在..\blog下。
每次修改本地文件后，需要hexo generate保存，hexo server先在本地查看，最后hexo deploy在网页查看。
hexo d提示ERROR: Deployer not found:git时，可执行npm install hexo-deployer-git --save再`hexo d。
博客搭建完成，可在http://yourgithubname.github.io查看。

Tips

hexo g == hexo generate //生成静态文件
hexo d == hexo deploy //部署文件
hexo s == hexo server //启动本地服务
hexo n == hexo new //新建博客
hexo new “post” //新建博客post
