java学习1-jdk和jre

发表于 2018-11-05 | 分类于 java

jdk与jre

jdk是java开发环境，用于开发java，包含jre；
jre为java运行环境，如果只要跑java程序，只需要安装jre即可；
jre：jvm+类库
jdk：jre+java编译打包等；

jdk中重要的包

bin：包括javac.exe-java编译程序，jar.exe-java打包程序等；
db：java数据库相关；
include：java的前身是c++，这里引入的是一些c、c++文件；
src.zip：java核心代码库，如lang、math等，以后要看java源代码，就从这里看；
jre：java运行环境；
