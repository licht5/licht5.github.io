Android Studio如何导入jar包

发表于 2017-04-09 | 分类于 java

今天做毕设的时候，需要把后台提供的json信息进行解析，用Gson比较方便。但是Gson包需要自己添加依赖，那么，怎么在Android Studio中导入jar包呢？

  1. 下载jar包
  2. 将下载的jar包直接粘贴到libs下（如果工程中没有，手动新建此libs文件夹）
  3. 在libs文件夹和添加的*.jar文件下鼠标单击菜单 add as library(由于我已经添加了，所以这里没有这个选项)
  4. 右击app选择Open Module Settings
  5. 在Dependencies中选择添加文件

    怎么看是否导入成功？
    打开App目录下有个build.gradle文件应该项目结构文件，如果有在文件里dependencies看到你的包名，那就是ok啦。
